Tutorial

Configuración de Paymentez

En métodos de pago de Sitio Web encontrarás Paymentez listo para configurar.

Puedes configurar las credenciales otorgadas por paymentez, mensajes predeterminados para el usuario
y de igual forma una configuración adicional para iconos de pagos permitidos y la cuenta de paymentez.

Realizar el pago desde la pantalla del cliente

Al momento de realizar el pago en nuestro sitio web, nos permitirá escoger el método de pago de paymentez
el cual mostrará los iconos de pagos permitidos y un mensaje de ayuda, los cuales son configurados desde el backend

Al dar click en pagar mostrrá el siguiente pop-up donde podemos ingresar los datos para el pago
como el nombre del títular, el número de tarjeta, código de seguridad y fecha de caducidad y podemos confirmar el pago

Una vez procesado el pago nos mostrará la información dl mismo y del pedido al igual que
el método de pago con el que se realizo la compra y un mensaje predeterminado configurado desde el backend
